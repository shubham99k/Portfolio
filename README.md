# Shubham Sali — Portfolio

Personal portfolio website built with React, Vite, and Tailwind CSS.

## Tech Stack

- **React 19** with Framer Motion
- **Vite 7** for bundling
- **Tailwind CSS 4** for styling
- **Vercel** for deployment

## Getting Started

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

## Deployment

Push to GitHub and connect the repo to [Vercel](https://vercel.com). It auto-detects Vite — no extra config needed.

## Project Structure

```
src/
├── components/    # All UI components
├── assets/        # Static assets
├── App.jsx        # Main app with scroll reveal
├── ThemeContext.jsx
├── index.css      # Theme variables & global styles
└── main.jsx       # Entry point
public/
├── favicon.svg    # Site icon
```
